# Proof 🚀

We are in the mission to bridges the gap between education and the job market by providing the participants with real, practical experience that transforms into verifiable proof of their skills.

---

## 🎯 Milestone: From Learning to Earning

- **Current Objective:** To empower **early-career professionals and graduates** to gain meaningful project experience that converts into tangible proof for recruitment and career growth.
- **Vision:** Evolving into a full-scale platform recognized globally as a standard for skill validation and talent discovery.

## 🛠 Tech Stack

We utilize a modern, scalable stack designed for high performance and a seamless user experience:

### Front-end

- **Next.js**: For optimized performance, SEO, and a robust React framework.
- **PWA (Progressive Web App)**: To provide a native app-like experience on mobile devices directly through the browser.
- **Tailwind CSS**: For rapid, responsive, and highly customizable UI development.

### Back-end

- **MongoDB**: A NoSQL database that allows for flexible data modeling and scalable growth as our user base expands.
